Raspy Studio 4.0.0

Olá internautas, meu nome é Pedro Nicolas sou o criador do Raspy Studio, hoje vim falar sobre a versão 4.0.0 que vai mudar todo o Raspy Studio! primeiramente quero falar que vai demorar para esta versão ser lançada, mas quando ela estiver pronta eu não vou lançar do nada, vou criar um programa beta para quem se inscrever vai ter acesso à versão e vai me ajudar a corrigir os bugs para que quando a versão estiver sem nenhum bug eu vou lança-la oficialmente.
Agora vou falar de recursos que eu quero colocar nesta nova versão como:

  • Autocompletar
  • Suporte para programar em java
  • Syntaxhighlight para outras linguagens
  • Nova área para gerenciar a pasta do seu projeto
E muito mais, e quando eu abrir o programa para testadores beta eu vou fazer uma postagem aqui no blog do Raspy Studio com umas imagens de como está a beta caso você não queira fazer parte do programa.
